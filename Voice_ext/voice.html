Close this tab - stops navigator

Available Voice Commands

  • chrome your_query - Asks Groq the query in relation to the webpage (e.g., "chrome what is on the page")
  • new tab - Open a new tab
  • close tab - Close the current tab
  • reload tab - Reload the current tab
  • next tab - Go to the next tab
  • previous tab - Go to the previous tab
  • go to tab [number] - Switch to a specific tab (e.g., "go to tab 3")
  • go back - Go back in browsing history
  • go forward - Go forward in browsing history
  • go to example.com - Navigate to a specific site
  • extensions - Open Chrome Extensions page
  • history - Open Chrome History
  • downloads - Open Chrome Downloads
  • scroll down - Scroll the page down
  • scroll up - Scroll the page up
  • google [query] - Search Google (e.g., "google OpenAI")
  • search [query] - Fill in a search field on the page
  • submit/enter - Submit the first form on the page
  • play video - Play a video on the page
  • stop listening - Stop Voice Navigator
  • close chrome - Close the entire Chrome window
